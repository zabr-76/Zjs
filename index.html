¿Qué es z-jackett.js?

Es un framework que permite enlazar datos y estructura en HTML usando atributos personalizados. Al actualizar los datos, la interfaz se cambia automáticamente sin intervención manual.

Funciones clave:

  • Reactividad profunda sobre objetos y arrays
  • Enlace dinámico de texto, HTML y atributos (z-text, z-html, z-bind)
  • Control condicional de visibilidad (z-show, z-if)
  • Listados reactivos con z-for
  • Sincronización fácil con inputs mediante z-model
  • Clases dinámicas con z-class o :class
  • Animaciones suaves con z-transition
  • Acceso directo a elementos con referencias ($refs) usando z-ref
